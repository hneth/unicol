# Bright colours of McGill University, Canada

`mcgill_brights` provides the bright colours of [McGill
University](https://www.mcgill.ca), Canada.

## Usage

``` r
mcgill_brights
```

## Format

An object of class `character` of length 6.

## Source

Colour definitions are based on [McGill](https://www.mcgill.ca)'s
[visual identity
guide](https://www.mcgill.ca/visual-identity/visual-identity-guide).

## Value

A named vector of colours (HEX/HTML codes of type character).

## Details

The bright colours are `"bright 1"` (defined as CMYK 0/50/100/0, RGB
247/148/29, HEX \#F7941D), `"bright 2"` (defined as CMYK 0/15/100/0, RGB
255/212/0, HEX \#FFD400), `"bright 3"` (defined as CMYK 70/0/30/0, RGB
39/189/190, HEX \#27BDBE), `"bright 4"` (defined as CMYK 60/0/0/0, RGB
68/200/245, HEX \#44C8F5), `"bright 5"` (defined as CMYK 35/0/100/0, RGB
178/210/53, HEX \#B2D235), and `"bright 6"` (defined as CMYK 20/72/0/0,
RGB 199/104/169, HEX \#C768A9).

`mcgill_brights` uses the HEX colour definition.

[McGill](https://www.mcgill.ca)'s secondary colours are not official,
but rather suggested colours that cover a larger spectrum to complement
the primary `"McGill red"`. They can be used to add variety and contrast
for headings, bullets, background and other graphic elements.

CMYK colour differs from RGB and HEX definition.
