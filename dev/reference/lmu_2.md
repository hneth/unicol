# Secondary colors of the LMU Munich, Germany

`lmu_2` provides four secondary colors of the
[Ludwig-Maximilians-Universität
München](https://www.lmu.de/en/index.html), Germany.

## Usage

``` r
lmu_2
```

## Format

An object of class `character` of length 4.

## Source

Color definitions are based on the [LMU brand
guide](https://www.lmu.de/de/die-lmu/struktur/zentrale-universitaetsverwaltung/kommunikation-und-presse/lmu-brand-guide/designgrundsaetze/farben/index.html).

## Value

A named vector of colors (HEX/HTML codes of type character).

## Details

The secondary colors of `lmu_2` are used to support the primary colors
of [`lmu_1`](https://hneth.github.io/unicol/dev/reference/lmu_1.md).

## Examples

``` r
lmu_2
#> dunkelgrau mittelgrau   hellgrau  lichtgrau 
#>  "#626468"  "#C0C1C3"  "#E6E6E7"  "#F5F5F5" 
unikn::seecol(lmu_2, main = "The secondary/grey colors of the LMU München")

unikn::demopal(lmu_2, type = 5, main = "Secondary/gray colors of LMU Munich")


# Combining primary and secondary LMU colors:
lmu_pal_bipolor <- c(lmu_1[-2], rev(lmu_2), lmu_1[2])
unikn::demopal(lmu_pal_bipolor, type = "mosaic", main = "Bipolar colors of LMU")


lmu_pal_linear <- c(lmu_1[-3], lmu_2, lmu_1[3])
unikn::demopal(lmu_pal_linear, type = "polygon", seed = 2, main = "Linear colors of LMU")

```
