# Colors of the University of Kassel, Germany

`uni_kassel` provides the eleven primary colors of the [University of
Kassel](https://www.uni-kassel.de/uni/), Germany.

## Usage

``` r
uni_kassel
```

## Format

An object of class `character` of length 11.

## Source

Color definitions are based on the [Uni
Kassel](https://www.uni-kassel.de/uni/)'s [CD und
Medienservice](https://www.uni-kassel.de/hochschulverwaltung/themen/toolbox-fuer-oeffentlichkeitsarbeit/)
page.

## Value

A names vector of colors (HEX/HTML codes of type character).

## Details

The 11 primary colors are `"k80"` (defined as HEX \#575757), `"k50"`
(defined as HEX \#9D9D9D), `"k20"` (defined as HEX \#DADADA),
`"hks28+k30"` (defined as HEX \#9A0C46), `"hks28"` (defined as HEX
\#C7105C), `"hks28-20"` (defined as HEX \#F3D8DD), `"dunkelgruen"`
(defined as HEX \#153824), `"hellblau"` (defined as HEX \#5095C8),
`"gelbgruen"` (defined as HEX \#C4D20F), `"hellgruen"` (defined as HEX
\#4AAC96), and `"goldgelb"` (defined as HEX \#EAC372).

`uni_kassel` uses the HEX color definitions.
