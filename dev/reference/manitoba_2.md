# Extended colors of the University of Manitoba, Canada

`manitoba_2` provides the 13 extended color palette colors of the
[University of Manitoba](https://umanitoba.ca/), Canada.

## Usage

``` r
manitoba_2
```

## Format

An object of class `character` of length 13.

## Source

Color definitions are based on the [UM-Brand Guidelines
(HTML)](https://umanitoba.ca/about-um/brand).

## Value

A named vector of colors (HEX/HTML codes of type character).

## Details

The 13 colors are `"PMS_2627"`, (defined as RGB 60/16/83, HEX \#3C1053),
`"PMS_294"` (defined as RGB 0/47/108, HEX \#002F6C), `"PMS_2199"`
(defined as RGB 0/187/220, HEX \#00BBDC), `"PMS_542"` (defined as RGB
123/175/212, HEX \#7BAFD4), `"PMS_560"` (defined as RGB 29/60/52, HEX
\#1D3C34), `"PMS_562"` (defined as RGB 0/111/98, HEX \#006F62),
`"PMS_3268"` (defined as RGB 0/171/142, HEX \#00AB8E), `"PMS_3255"`
(defined as RGB 44/213/196, HEX \#2CD5C4), `"PMS_430"` (defined as RGB
124/135/142, HEX \#7C878E), `"PMS_416"` (defined as RGB 126/127/116, HEX
\#7E7F74), `"PMS_158"` (defined as RGB 232/119/34, HEX \#E87722),
`"PMS_129"` (defined as RGB 243/208/62, HEX \#F3D03E), and `"warm_gray"`
(defined as RGB 131/120/111, HEX \#83786F).

`manitoba_2` uses the HEX color definitions.
