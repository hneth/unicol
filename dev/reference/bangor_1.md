# Primary corporate colours of the University of Bangor, Wales

`bangor_1` provides the four primary corporate colours of the
[University of Bangor](https://www.bangor.ac.uk), Wales, UK.

## Usage

``` r
bangor_1
```

## Format

An object of class `character` of length 4.

## Source

Colour definitions are based on [Bangor](https://www.bangor.ac.uk)'s
Brand guidelines (PDF).

## Value

A named vector of colors (HEX/HTML codes of type character).

## Details

`bangor_1` is based on HEX color definitions (see
[Bangor](https://www.bangor.ac.uk)'s Brand guidelines, PDF, p. 10, for
names and details).
