# Primary colors of the University of Freiburg, Germany

`uni_freiburg_1` provides a primary color palette of the [University of
Freiburg](https://uni-freiburg.de/), Germany.

## Usage

``` r
uni_freiburg_1
```

## Format

An object of class `character` of length 7.

## Source

Color definitions are based on the new web style guide at
<https://uni-freiburg.de/> (at `https://cd.uni-freiburg.de/farben/` on
2023-04-28).

## Value

A named vector of colors (HEX/HTML codes of type character).

## Details

The primary color `blau` (here: `uni_freiburg_1[1]`) is alternatively
defined as `Pantone 7687C`, `RGB 52/74/154`, `HEX #344a9a`, or
`CMYK 90/75/0/0`.

The color `sand` is alternatively defined as `HEX #f6f1e3`,
`RGB 246/241/227`, and `CMYK 0/2/10/5`.

The gradient of `blau` colors is defined on
<https://cd.uni-freiburg.de/farben/>.

See
[`uni_freiburg_0`](https://hneth.github.io/unicol/dev/reference/uni_freiburg_0.md)
for the basic colors (without the `Blau` color gradient) and the
[`ac`](https://rdrr.io/pkg/unikn/man/ac.html) and
[`usecol`](https://rdrr.io/pkg/unikn/man/usecol.html) functions for
creating alternative color gradients.

## Examples

``` r
uni_freiburg_1
#>       blau   blau 80%   blau 60%   blau 40%   blau 20% dunkelblau       sand 
#>  "#344A9A"  "#5D6BAD"  "#868DC2"  "#AFB1D8"  "#D7D8EC"  "#00004a"  "#F6F1E3" 
unikn::seecol(uni_freiburg_1, main = "Primary color gradient of the University of Freiburg")

unikn::demopal(uni_freiburg_1, type = 1, main = "Primary color gradient of Freiburg University")


# Get a "sand" color gradient:
sand_gradient <- unikn::usecol(c(uni_freiburg_1["sand"], "white"), n = 3)[1:3] 
# demopal(sand_gradient)
```
