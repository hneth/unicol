# Primary colors of the Martin Luther University Halle-Wittenberg, Germany

`uni_halle` provides the colors of the [Martin Luther University
Halle-Wittenberg](https://example.org) (MLU), Germany.

## Usage

``` r
uni_halle
```

## Format

An object of class `character` of length 8.

## Source

Color definitions are based on the [MLU](https://www.uni-halle.de)'s
[Corporate design](https://www.marketing.uni-halle.de/corporate-design/)
specifications (PDF).

## Value

A named vector of colors (HEX/HTML codes of type character).

## Details

`uni_halle` contains 1 primary color `MLU gruen` and 7 faculty-specific
colors.

`uni_halle` is based on the online/screen versions of HEX color
definitions.

``` r

#>              "#b1e053"              "#2d7aab"              "#368429" 
#> Philosophie I-III lila      Medizin rotorange    Jura Wiwi cranberry 
#>              "#993c8c"              "#cb472a"              "#aa263c" 
#>      Theologie schwarz         Theologie grau 
#>              "#000000"              "#424242" 
unikn::seecol(uni_halle, main = "Colors of the MLU Halle-Wittenberg, Germany")  # view palette

```
