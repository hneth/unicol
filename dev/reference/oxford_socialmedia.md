# Social Media colours of University of Oxford, England, UK

`oxford_socialmedia` provides the eleven social media colours of the
[University of Oxford](https://www.ox.ac.uk/), England, UK.

## Usage

``` r
oxford_socialmedia
```

## Format

An object of class `character` of length 11.

## Source

Colour definitions are based on the [Digital style
guide](https://www.ox.ac.uk/public-affairs/style-guide/digital-style-guide).

## Value

A named vector of colours (HEX/HTML codes of type character).

## Details

The 11 social media colours are `"cherry"` (defined as HEX \#be0f34),
`"purple"` (defined as HEX \#ac48bf), `"red"` (defined as HEX \#c7302b),
`"light_blue"` (defined as HEX \#1daced), `"deep_blue"` (defined as HEX
\#001c3d), `"lagoon_blue"` (defined as HEX \#006599), `"weibo"` (defined
as HEX \#d62a2a), `"green"` (defined as HEX \#177770), `"orange"`
(defined as HEX \#fb8113), `"google_plus"` (defined as HEX \#d34836),
and `"instagram"` (defined as HEX \#517fa4),
