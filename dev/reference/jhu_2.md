# Secondary colors of Johns Hopkins University (JHU), USA

`jhu_2` provides the six secondary colors of the [JHU, Johns Hopkins
University](https://www.jhu.edu), USA.

## Usage

``` r
jhu_2
```

## Format

An object of class `character` of length 6.

## Source

Color definitions are based on the [Color
manual](https://brand.jhu.edu/color/).

## Value

A named vector of colors (HEX/HTML codes of type character).

## Details

The secondary color palette is designed to provide creative flexibility.
These colors are complementary to the primary blues and were chosen to
add warmth and energy to your design.

These secondary colors are `"JHU red"` (defined as PMS 173 C, CMYK
0/82/94/2, RGB 207/69/32, HEX \#CF4520), `"JHU burgundy 1"` (defined as
PMS 188 C, CMYK 16/100/65/58, RGB 118/35/47, HEX \#76232F),
`"JHU purple 1"` (defined as PMS 7655 C, CMYK 33/72/0/0, RGB 161/90/149,
HEX \#A15A95), `"JHU green 1"` (defined as PMS 3278 C, CMYK 99/0/69/0,
RGB 0/155/119, HEX \#009B77), `"JHU seablue"` (defined as PMS 285 C,
CMYK 90/48/0/0, RGB 0/114/206, HEX \#0072CE), and `"JHU yellow"`
(defined as PMS 7406 C, CMYK 0/20/100/2, RGB 241/196/0, HEX \#F1C400).
