# Darker pastel colours of the University of Lund, Sweden

`lund_3` provides darker tints of the five pastel colours of the [Lund
University](https://www.lu.se), Sweden.

## Usage

``` r
lund_3
```

## Format

An object of class `character` of length 5.

## Source

Color definitions are based on [LU](https://www.lu.se)'s staff
guidelines on
[Colour](https://www.staff.lu.se/support-and-tools/communication-and-graphic-profile/graphic-profile-and-logotype/colours)
(HTML).

## Value

A named vector of colors (HEX/HTML codes of type character).

## Details

See [`lund_2`](https://hneth.github.io/unicol/dev/reference/lund_2.md)
for the pastel colours of Lund University.

`lund_3` is based on HEX color definitions.
