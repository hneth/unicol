# Gradient colors of the University of Bonn, Germany

`uni_bonn_2` provides 4 shades (of 100, 75, 50, and 25 percent) for each
of the 3 primary colors of the [University of
Bonn](https://www.uni-bonn.de/en), Germany.

## Usage

``` r
uni_bonn_2
```

## Format

An object of class `character` of length 12.

## Source

Color definitions are based on the [CD
manual](https://www.uni-bonn.de/de/universitaet/presse-kommunikation/corporate-design).

## Value

A named vector of colors (HEX/HTML codes of type character).

## Details

See
[`uni_bonn_1`](https://hneth.github.io/unicol/dev/reference/uni_bonn_1.md)
for the definitions of the three primary colors and the
[`ac`](https://rdrr.io/pkg/unikn/man/ac.html) and
[`usecol`](https://rdrr.io/pkg/unikn/man/usecol.html) functions for
creating color gradients.

## Examples

``` r
uni_bonn_2
#>     uni blau uni blau 75% uni blau 50% uni blau 25%     uni gelb uni gelb 75% 
#>    "#004E9F"    "#3F7AB7"    "#7FA6CF"    "#BFD2E7"    "#FCBA00"    "#FCCB3F" 
#> uni gelb 50% uni gelb 25%     uni grau uni grau 75% uni grau 50% uni grau 25% 
#>    "#FDDC7F"    "#FEEDBF"    "#909085"    "#ABABA3"    "#C7C7C2"    "#E3E3E0" 
unikn::seecol(uni_bonn_2, main = "Gradient colors of the University of Bonn")  # view color palette 

unikn::demopal(uni_bonn_2, type = 3, main = "Color shades of the University of Bonn")

```
