# Primary colours of the Lancaster University, England, UK

`lancaster_1` provides the three Primary colours of [Lancaster
University](https://www.lancaster.ac.uk), England, UK.

## Usage

``` r
lancaster_1
```

## Format

An object of class `character` of length 3.

## Source

Colour definitions are based on [Lancaster's Primary guide
(PDF)](https://www.lancaster.ac.uk/media/lancaster-university/content-assets/documents/current-staff/brand/LUBrandGuidelines-Staff-Nov-2018.pdf).

## Value

A named vector of colours (HEX/HTML codes of type character).

## Details

The three primary colours of [Lancaster
University](https://www.lancaster.ac.uk) are `"Pantone 1807"` (defined
as HEX: \#b5121b) `"Pantone 429"` (defined as HEX: \#bec0c2), and
`"Pantone 432"` (defined as HEX: \#555656).

`lancaster_1` uses the HEX colour definitions.
