# Black color gradient of the Free University (FU) Berlin, Germany

`fu_2` provides the black color gradient of the [Free University
Berlin](https://www.fu-berlin.de/en/index.html), Germany.

## Usage

``` r
fu_2
```

## Format

An object of class `character` of length 11.

## Source

Color definitions are based on the
[FU](https://www.fu-berlin.de/en/index.html)'s
[CI](https://www.fu-berlin.de/sites/corporate-design/) page.

## Value

A named vector of colors (HEX/HTML codes of type character).

## Details

The color values were updated in 2024.
