# Shade colours of University of Oxford, England, UK

`oxford_shades` provides the seven shade colours of the [University of
Oxford](https://www.ox.ac.uk/), England, UK.

## Usage

``` r
oxford_shades
```

## Format

An object of class `character` of length 7.

## Source

Colour definitions are based on the [Digital style
guide](https://www.ox.ac.uk/public-affairs/style-guide/digital-style-guide).

## Value

A named vector of colours (HEX/HTML codes of type character).

## Details

The 7 shade colours are `"black"` (defined as HEX \#000000),
`"off_black"` (defined as HEX \#333333), `"mid_grey"` (defined as HEX
\#7a736e), `"dark_grey"` (defined as HEX \#a6a6a6), `"light_grey"`
(defined as HEX \#e0ded9), `"light_grey"` (defined as HEX \#f9f8f5), and
`"white"` (defined as HEX \#ffffff).
