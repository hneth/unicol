# Accent colors of the LMU Munich, Germany

`lmu_3` provides five accent colors of the
[Ludwig-Maximilians-Universität
München](https://www.lmu.de/en/index.html), Germany.

## Usage

``` r
lmu_3
```

## Format

An object of class `character` of length 5.

## Source

Color definitions are based on the [LMU brand
guide](https://www.lmu.de/de/die-lmu/struktur/zentrale-universitaetsverwaltung/kommunikation-und-presse/lmu-brand-guide/designgrundsaetze/farben/index.html).

## Value

A named vector of colors (HEX/HTML codes of type character).

## Details

The accent colors of `lmu_3` are used sparsely in combination with the
primary colors of
[`lmu_1`](https://hneth.github.io/unicol/dev/reference/lmu_1.md) and the
secondary colors of
[`lmu_2`](https://hneth.github.io/unicol/dev/reference/lmu_2.md).

``` r
unikn::demopal(lmu_3, type = 3, main = "Accent colors of LMU Munich")

```
